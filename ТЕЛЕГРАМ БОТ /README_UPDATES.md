# Обновления (Fixes)

## 1. Скроллинг Mini App
**Файл:** `index.html`
**Изменения:** 
- Увеличена высота экрана до `100dvh` (для мобильных).
- Добавлен `padding-bottom: 150px` для контейнера `product-view`.
- Добавлен `overscroll-behavior: contain`.

**⚠️ ВАЖНО:** 
Чтобы эти изменения заработали у пользователей, вы должны **загрузить обновленный файл `index.html` на ваш хостинг** (GitHub Pages и т.д.). Телеграм не берет этот файл с вашего компьютера, он берет его по ссылке `WEB_APP_URL`.
