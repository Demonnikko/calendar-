# Инструкция по созданию Telegram-бота

Чтобы наш бот заработал, его нужно сначала зарегистрировать в самом Telegram. Для этого используется специальный бот — **@BotFather**.

## Шаги для регистрации:

1.  **Найдите @BotFather в Telegram:**
    *   Откройте Telegram и в строке поиска введите `@BotFather`.
    *   Выберите официального бота с синей галочкой верификации.

2.  **Создайте нового бота:**
    *   Начните диалог с BotFather, нажав кнопку "Start" или отправив команду `/start`.
    *   Отправьте команду `/newbot`, чтобы начать процесс создания нового бота.

3.  **Выберите имя и юзернейм:**
    *   **Имя:** BotFather попросит вас придумать имя для бота. Это имя будут видеть ваши пользователи в диалоге. Например: `Illusionist OS Store`.
    *   **Юзернейм (username):** Затем BotFather попросит задать уникальный юзернейм. Он должен заканчиваться на `bot`. Например: `IllusionistOS_bot` или `IllusionistStoreBot`.

4.  **Получите API-токен:**
    *   После успешного создания бота BotFather пришлет вам сообщение, в котором будет содержаться **токен** — уникальный ключ для доступа к API Telegram.
    *   Этот токен выглядит как длинная строка из цифр и букв, например: `1234567890:ABCdEfGhIjKlMnOpQrStUvWxYz-123456789`.

**Важно:** Этот токен — секретный. Не передавайте его никому и не публикуйте в открытом доступе.

## Что делать дальше?

Как только вы получите токен, **пожалуйста, пришлите его мне в следующем сообщении**. Он понадобится мне для того, чтобы оживить вашего бота и продолжить разработку.
